# 100Hires Assessment - Research Project: AI-Powered SEO Content Production

**Candidate:** Pham Le Anh Vu
**Role:** Junior Growth Marketing Specialist (AI-Native)

## 1. Project Overview
This repository contains a curated research dataset focusing on **AI-powered SEO content production**. 

Instead of scraping generic advice, I focused on extracting high-signal, actionable workflows from 10 top-tier practitioners who are actively testing and breaking AI search algorithms.

## 2. What I Collected
Using Cursor IDE alongside Claude Code / Codex, I automated the extraction of real-world insights:
* **`/research/youtube-transcripts/`**: Successfully used AI automation to fetch and format transcripts from 4 comprehensive YouTube guides, bypassing the need for manual Python execution when environmental issues arose.
* **`/research/linkedin-posts/`**: Scraped and formatted recent, high-value LinkedIn posts from 6 industry leaders discussing recent Google Core Updates and AI Overviews.
* **`/research/sources.md`**: A clean directory of all 10 experts with direct links and brief annotations explaining their specific niche in AI SEO.

## 3. Why I Chose These Experts
I selected these specific voices because they "practice what they teach":
* **The Testers (Julian Goldie, Matt Diggity):** They run large-scale programmatic SEO tests and conduct rigorous case studies comparing AI vs. human content.
* **The Technical Strategists (Koray Tuğberk GÜBÜR, Gael Breton):** They focus on advanced concepts like Semantic SEO, topical maps, and practical API applications for building niche sites.
* **The Macro-Thinkers (Aleyda Solis, Kevin Indig, Eli Schwartz):** They provide the high-level strategy needed to adapt to Google's Search Generative Experience (SGE) and shifting B2B search intent.
* **The Analysts (Steve Toth, Mark Williams-Cook, Lily Ray):** They bridge the gap between AI generation and Google's strict E-E-A-T guidelines, ensuring content ranks safely.

## 4. Technical Workflow & Challenges Overcome
* Leveraged AI coding agents within Cursor to normalize Markdown formatting across all gathered files.
* Bypassed local Python environment limitations (`pip` module errors) by prompting the AI Agent to utilize its own web-browsing capabilities to fetch YouTube transcripts directly, demonstrating a core AI-Native trait: **Resourceful problem solving**.
